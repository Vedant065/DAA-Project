# MST Visualizer

Streamlit app to visualize Kruskal's and Prim's Minimum Spanning Tree algorithms.

## Run locally

```
pip install -r requirements.txt
streamlit run streamlit_mst_visualizer.py
```

## Features
- Add/delete nodes and edges
- Visual step-through of Kruskal & Prim
- BFS/DFS traversal
- Shortest path visualization
