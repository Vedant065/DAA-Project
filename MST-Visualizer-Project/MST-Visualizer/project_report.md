# Project Report: MST Visualizer

## Objective
Create an interactive Streamlit tool to visualize the formation of Minimum Spanning Trees using Kruskal and Prim algorithms.

## Tools Used
- Python
- Streamlit
- NetworkX
- Matplotlib

## Key Features
- Graph editing (add/remove nodes/edges)
- Step-by-step algorithm visualization
- BFS/DFS traversal
- Shortest path highlighting
